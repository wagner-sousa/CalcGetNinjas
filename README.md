# CalcGetNinjas

Calculadora de custos para freelancers da plataforma GetNinjas. Permite calcular o custo por moeda e o custo de atividades em reais (BRL).

## Tech

Vue 3 + TypeScript + Vite + Vitest + Docker

## Comandos

```bash
docker compose run --rm app npm run dev
docker compose run --rm app npm run test
docker compose run --rm app npm run build
```
